GaugeMeter

    gauge chart

    결과화면

    HTML

    <script src="../resource/chart/gauge/gaugemeter.js"></script>
     
    <div class="gauge_meter" data-percent="60" data-label="Percent" data-size="188"></div>
    		 cs

    Option

    • data-percent : 게이지 설정 값
    • data-style : 기본은 원, Arch : 아치형, semi : 반원
    • data-width : 게이지 두께
    • data-label : 중앙의 설정 값, 텍스트 아래 보조 텍스트
    • data-size : 차트 크기 설정 ( px 단위 )
    • data-color : 게이지 진행률 막대 색상 ( 디폴트 - #2C94E0 // data-theme가 지정된 경우 무시됨 )/ data-back : 게이지 배경색상 ( 디폴트 - RGBa (0,0,0, .06) )
    • data-theme : 게이지 진행률 막대 색상 및 그라데이션 (게이지 값의 변화에 따라 색상 변경 // 디폴트 : Red-Gold-Green)
    • data-stripe : 게이지 진행률 막대를 단색 또는 스트라이프 형식으로 표시, 값이 0보다 클 경우 스트라이프로 변경되며 해당 값이 두께
