    옵션 관련

    • apexcharts 홈페이지 옵션 페이지에서 각 옵션 value값을 잘 확인해서 넣어야 함.

      colors: Array /////// colors: Single Colors || Array of Colors

    • IE/Edge에서는 toolbar(다운로드)기능이 정상작동 안함.
    • 단독 colors값을 따로 지정을 안하면 apexchart의 기본 색상으로 지정됨.
    • 차트 데이터 변경시 : updateSeries // 차트 옵션 변경시 : updateOptions 사용 
      // 차트 데이터 변경
      dayChart4.updateSeries([{

       },{
