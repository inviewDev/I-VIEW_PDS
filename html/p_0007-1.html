    Scroll Indicator

    info

    • CSS는 프로젝트 상황에 맞게 변경해서 사용할 것.

    결과화면

    테스트 배경이미지

    html

    <div class="scroll-line"></div>
    		cs

    CSS

    .scroll-line {position: fixed; top: 0; left: 0; height: 7px; background-color: #5567ff}
    		cs

    JS

    $(window).scroll(function(){
    var winTop = $(window).scrollTop(),
        docH = $(document).height(),
        winH = $(window).height();
     
                var scrolled = (winTop/(docH-winH))*100;
            $('.scroll-line').css('width', (scrolled + '%'));
    });
